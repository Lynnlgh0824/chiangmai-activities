## 🚀 使用方法

### 快速开始

1. **访问仪表板**
   ```
   http://localhost:3000/test-automation-dashboard.html
   ```

2. **点击"运行所有测试"**
   - 自动运行所有测试套件
   - 实时显示进度
   - 完成后显示总结

3. **查看详细结果**
   - 点击测试套件展开详情
   - 查看每个测试点的结果
   - 阅读错误消息（如果有）

4. **查看日志**
   - 点击"查看日志"按钮
   - 实时日志输出
   - 彩色编码显示
