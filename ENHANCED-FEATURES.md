## 🔧 技术实现

### 核心技术
- **纯 HTML**: 单文件，无需构建
- **Vanilla JS**: 原生 JavaScript，无框架
- **CSS3**: 现代样式和动画
- **Fetch API**: 数据获取

### 数据流
```
用户操作 → JavaScript 筛选 → 更新 DOM
                ↓
         fetch API 调用
                ↓
         后端 Express API
                ↓
         JSON 数据返回
```

### 性能优化
- ✅ 懒加载（只在需要时获取数据）
- ✅ 内存缓存（allActivities 变量）
- ✅ 最小化 DOM 操作
- ✅ CSS 动画（GPU 加速）
