## 📋 问题识别

### 用户需求

### 问题分析

**提示位置**: 第5093行
```javascript

    // ...
    showAutoSelectToast(day); // ← 这里产生提示

```

**提示样式**: 第5109-5159行
```javascript
function showAutoSelectToast(day) {
    const toast = document.createElement('div');
    toast.className = 'h5-auto-select-toast';
    toast.innerHTML = `✨ 已自动选中 ${dayNames[day]}`;
    // ... 显示2秒后自动消失

```

**触发场景**:
- 移动端（≤768px）
- 周视图模式
- 未手动选择日期时

### ✅ 自动选中功能保留

- ✅ 日期标签仍会自动高亮

```
用户滚动 → 自动选中视野中的日期

```
