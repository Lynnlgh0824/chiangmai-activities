## 🐍 转换脚本

### excel-to-json.py ⭐

**功能**：将飞书Excel数据转换为项目JSON格式

```bash
# 基本用法
python3 excel-to-json.py 清迈活动数据.xlsx

# 指定输出文件
python3 excel-to-json.py 清迈活动数据.xlsx data/items.json
```

**支持格式**：
- ✅ Excel (.xlsx, .xls)
- ✅ CSV (.csv)

**特性**：
- 自动检测文件格式
- 智能字段映射
- 实时进度显示
- 完善的错误处理

## 🆘 常见问题

### Q: 前端无法访问？
**A:** 检查服务器是否启动
```bash
# 检查进程
ps aux | grep node

# 重新启动
npm run dev
```

### Q: 转换脚本报错？
**A:** 检查依赖库
```bash
pip3 install pandas openpyxl
```

### Q: 数据未更新？
**A:**
1. 确认Excel文件格式正确
2. 检查转换脚本是否成功
3. 强制刷新浏览器 (Ctrl+Shift+R)
