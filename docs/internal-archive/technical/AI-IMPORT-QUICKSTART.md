## 📝 完整示例

### 步骤 1: 复制文本

在小红书看到活动：
```
清迈流瑜伽早课
时间：每周一三五 9:00-11:00
地点：宁曼路瑜伽馆
价格：500฿/节
```

复制这段文本（Ctrl+C）

### 步骤 2: 粘贴解析

1. 打开 `ai-import.html`
2. 粘贴到输入框（Ctrl+V）
3. 点击 **"🤖 AI 智能解析"**

### 步骤 3: 查看结果

AI 自动解析：
- ✅ 标题：清迈流瑜伽早课
- ✅ 类型：固定频率活动
- ✅ 星期：周一,周三,周五
- ✅ 时间：09:00-11:00
- ✅ 地点：宁曼路瑜伽馆
- ✅ 价格：500฿

### 步骤 4: 导出保存

1. 点击 **"📊 导出为 JSON"**
2. 下载文件到 `scraper` 目录
3. 运行命令：
   ```bash
   cd scraper
   node excel-writer-import.js ai-import-1234567890.json
   ```

4. 完成！数据已写入 `清迈活动数据.xlsx`

## 🎯 核心功能

### ✨ AI 自动识别

- ✅ 活动标题
