## ⚠️ 常见问题

### Q1: 我应该用哪个地址？

**A**:
- **本机测试**: 用 `http://localhost:3000`
- **手机测试**: 用 `http://192.168.1.133:3000`

### Q2: 为什么有两个地址？

**A**: 这不是两个不同的应用，而是**同一个应用**的两种访问方式：
- `localhost` - 本机内部访问
- `192.168.1.133` - 局域网外部访问

### Q3: 地址打不开怎么办？

**A**: 检查服务器是否运行：
```bash
lsof -i :3000
```

如果没有输出，说明服务器未运行，需要启动：
```bash
cd /Users/yuzhoudeshengyin/Documents/my_project/Chiengmai
npm start
```
